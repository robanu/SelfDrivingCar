# SelfDrivingCar
Components used:
  1. NVIDIA Jetson Nano
  2. Arduino Nano
  3. USB Webcam
  4. Ultrasonic sensors HC04
  5. 12V DC motors
  6. Wheels, Chassis, Separators
  7. Motor Driver L298
  8. SD Card for Jetson Nano
  9. Wi-Fi adapter for Jetson Nano
